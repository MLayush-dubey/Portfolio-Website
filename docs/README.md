# docs/

Project documentation and design reference. None of this is part of the app build — it
lives outside `app/` so Next.js never compiles or ships it.

| Path | What it is |
|---|---|
| `PLAN.md` | Original build plan / spec for porting the design into this Next.js app. |
| `AUDIT.md` | Gap audit of the implementation against the design spec. |
| `design-reference/` | The original "Latent Space" design prototype — `Portfolio.html` plus Babel/JSX component mockups in `js/`. Visual source of truth, **reference only, not imported by the app**. See `design-reference/README.md` for the full handoff. |
